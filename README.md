# Sentinel API Analysis Tool

A powerful tool for analyzing Sentinel-2 satellite data, focusing on Land Cover Classification and NDRE/NDVI vegetation indices.

## Features
- **Automated Data Retrieval**: Downloads Sentinel-2 imagery for specified regions and years.
- **Vegetation Analysis**: Calculates NDVI and NDRE to assess vegetation health.
- **Land Cover Classification**: Classifies terrain into Forest, Agriculture, Deforested, and other categories.
- **Change Detection**: Compares data between years to analyze environmental changes.
- **Interactive Web App**: Visualizes results on a local web dashboard.

## How to Run

Double-click `run_sentinel.bat` to start the application. 
OR 
Run the following command in your terminal:
```bash
python api.py
```

### Linux / Mac / GitHub Codespaces
Run:
```bash
./run_sentinel.sh
```

**Do not run .gitignore**.
